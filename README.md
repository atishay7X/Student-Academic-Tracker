# Student-Academic-Tracker
Student Academic Tracker Excel Project

Built a 5-sheet Excel Academic Tracker with 193 live formulas tracking marks, attendance, assignments, and CGPA across 6 subjects for Class 10


Implemented weighted score calculation using nested IF formulas based on CBSE exam weightage (UT1 10%, Mid-Term 20%, UT2 10%, Final 60%)


Created VLOOKUP and INDEX-MATCH lookup engine to retrieve subject-wise performance data dynamically from a reference table


Designed a Pivot-style summary using SUMIF and AVERAGEIF to analyse marks by subject and exam type, replicating Excel Pivot Table output


Built 3 chart types (Clustered Bar, Line, Pie) to visualise marks trends, exam comparisons, and score distribution across subjects


Applied conditional formatting rules (Color Scale, CellIs) to auto-flag attendance below 75% and highlight pass/fail status without manual review


Used cross-sheet references and TODAY() function to build a live assignment tracker that auto-calculates days remaining and flags overdue tasks


Designed an executive Dashboard pulling KPIs from 4 sheets using direct cell references and INDEX-MATCH, creating a single-view summary of all academic data
